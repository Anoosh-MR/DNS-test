# DNS-test

## Installation

Install my-project with npm

```bash
  git clone https://github.com/Anoosh-MR/DNS-test.git
```

```bash
  cd DNS-test
```

```bash
  cd server
  npm install
  npm start
```

```bash
  cd client
  npm install
  npm run dev
```

---

     Add .env file in server folder with

```bash
  PORT=
  MONGO_URL=
```
